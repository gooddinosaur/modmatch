# ModMatch

ModMatch is a high-performance car parts marketplace designed to solve part fitment and buyer protection problems.

## Overview

ModMatch helps car owners find parts that are guaranteed to fit their specific vehicles.
The platform includes an admin verification process and a safe payment workflow that holds funds until buyers confirm the part is correct.

## Key Goals

* Accurate part-to-vehicle matching

* Protection against fake or incorrect parts

* Secure transaction workflow (simulated escrow system)

## Tech Stack (Planned)

* Frontend: Next.js (npm run dev)

* Backend: FastAPI (Python) (uvicorn app.main:app --reload)

* Database: PostgreSQL
